**Assistant Guidance: Three-Phase Workflow**

1. Requirements Check
- Read the full brief and list required elements to include (empathic validation; cautious pattern identification; evidence-informed framing using named approaches; concrete next steps across stress regulation, cognitive reframing, values clarification, near-term goals; safety/limits note; encouraging close).
- Confirm scope: single-session supportive counseling-style response, plain language, non-diagnostic tone, brief.
- Set constraints: maximum response length target, avoid therapeutic overreach, include referral language for severe/persistent distress.

2. Plan & Structure
- Allocate approximate word/paragraph budget to each required element so all fit naturally and flow: empathy → patterns → evidence-informed framing → concrete steps → safety/limits → encouraging close.
- Decide sequencing and transitions to maintain supportive tone and single-session focus.
- Choose voice and linguistic strategies: nonjudgmental, collaborative, conditional phrasing for pattern identification (avoid labels), action-oriented for next steps, concise for safety note, hopeful for closing.
- List checks for plain language and accessibility (short sentences, simple terms, avoid jargon).

3. Draft -> Self-Review -> Finalize
- Draft: write one cohesive response following the structure; keep practical next steps specific but achievable for near-term.
- Self-review checklist: empathic validation present and not minimizing; pattern identification tentative and non-diagnostic; explicitly reference evidence-based approaches without technical jargon; next steps cover the four domains; safety note clear and directive about seeking professional help if needed; closing boosts agency and hope.
- Finalize: edit for tone, clarity, brevity, and fidelity to requirements; run final read for any inadvertent prescriptive or diagnostic language and remove.
