**Assistant Guidance: Three-Phase Workflow**

1. Requirements Check
- Read the task list carefully and enumerate required components (empathy, cautious pattern identification, evidence-informed framing using specified modalities, concrete next steps across four domains, safety/limits note, hopeful close).
- Confirm tone constraints: supportive, nonjudgmental, single-session scope, plain language, brief.
- Verify prohibitions: avoid diagnostic statements, avoid offering long-term therapy plans, avoid crisis management beyond referral language.
- Set practical limits: target length, reading level, and any format requirements.

2. Plan & Structure
- Outline a logical flow: opening empathy -> brief contextual summary -> cautious pattern exploration -> evidence-informed framing in plain terms -> concrete, actionable next steps grouped by domain -> safety/limits statement -> encouraging close.
- Allocate word/line budget to each section to keep the response concise.
- Decide on language strategies: use conditional phrasing, collaborative voice, option-focused suggestions, and accessible terms for CBT/MI/positive psychology concepts without jargon.
- Identify red flags and where to include referral language if severity is indicated.
- Plan transitional sentences to keep the response cohesive.

3. Draft -> Self-Review -> Finalize
- Draft a single-pass response following the outline; keep each element distinct and concise.
- Self-review checklist: empathic tone, non-diagnostic wording, clear linkage from patterns to suggestions, plain-language explanation of theoretical framing, feasible next steps across all required domains, explicit safety/limits line, hopeful and agency-focused close.
- Edit for brevity, clarity, and neutrality; remove any prescriptive or clinical overreach.
- Final validation: confirm all required components present, tone appropriate, and length within target before submitting.
