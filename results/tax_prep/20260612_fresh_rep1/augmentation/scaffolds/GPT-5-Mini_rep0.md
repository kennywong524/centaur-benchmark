**Assistant Guidance: Three-Phase Workflow**

1) Requirements Check
- Inventory all supplied documents and pre-filled return pages; mark any inconsistencies, omissions, and mismatches between source documents and return entries.
- Create a targeted missing-data list (e.g., expense detail, support documentation, dates, withholding slips) that prevents exact liability calculation.
- Identify which filing-status, dependency, and income items require eligibility analysis or additional proof.
- Flag all potentially affected federal and state lines and note where the state return copies federal figures.

2) Plan & Structure
- For each flagged item, define the analytical task: verify source value → reconcile to return → determine correct code/line → identify required form/schedule.
- Specify which federal and state rules need review (eligibility tests, deduction vs standard choice, self-employment tax, withholding credits, etc.) and list reference points to consult.
- Establish assumptions to use when information is incomplete and document how those assumptions alter calculations.
- Outline the sequence of recalculation steps: correct income items, recompute AGI, reassess deductions/credits, recalc tax and state adjustments, then reflow withholding and net liability.
- Prepare a mapping of corrections to specific forms/schedules that will need amendment.

3) Draft -> Self-Review -> Finalize
- Draft clear, neutral explanations for each discrepancy, the applicable rule(s) consulted, and the correction method used; include a labeled estimate framework when exact figures are unavailable.
- Perform a structured self-review: check arithmetic, confirm consistency across federal/state returns, verify form line references, and ensure assumptions are explicitly noted.
- Finalize deliverables: corrected form list, recalculation summary with assumption log, items needed from client, and recommended next steps for amendment or documentation.
