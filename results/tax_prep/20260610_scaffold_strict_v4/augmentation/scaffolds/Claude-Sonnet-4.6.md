**Assistant Guidance: Three-Phase Workflow**

**Phase 1: Requirements Check**

Before any analysis begins, catalog all inputs and flag gaps:
- Inventory every client-provided figure and every pre-filled return entry as two separate lists.
- Identify explicitly missing data (e.g., freelance business expenses) and note that these gaps will prevent exact liability calculation—plan to articulate what is missing and why it matters.
- Confirm which tax year and jurisdiction govern, and note any jurisdiction-specific rules that will need to be applied (without yet applying them).
- Flag each pre-filled entry as "verify" rather than accepting it as correct.

**Phase 2: Plan & Structure**

Map out the logical sequence of issues to address before writing anything:
- Group discrepancies by category (income errors, missing schedules, deduction method, filing status, dependent eligibility, state return linkage).
- For each category, identify the relevant form or schedule implicated.
- Determine the order of dependencies (e.g., corrected income affects AGI, which affects itemized vs. standard deduction comparison, which affects state return).
- Note where missing information requires a conditional framework rather than a firm number.
- Outline the response structure so each issue is addressed systematically without overlap.

**Phase 3: Draft → Self-Review → Finalize**

- Draft each section following the planned structure, treating every pre-filled figure as unverified.
- Self-review: check that each discrepancy is explained with its rule basis, each missing item is named, and no conclusion assumes unavailable data.
- Verify internal consistency (corrected figures flow correctly through dependent calculations, AGI, and state return).
- Finalize by confirming the response distinguishes between what can be calculated, what requires estimation, and what requires additional client input.
