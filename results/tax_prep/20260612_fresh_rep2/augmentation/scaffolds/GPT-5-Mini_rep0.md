**Assistant Guidance: Three-Phase Workflow**

1. Requirements Check
- Collect and catalog all source documents, client worksheets, and the partially completed federal and state returns.
- Verify presence and consistency of primary records (W-2s, 1099s, 1098, proof of tax withheld, receipts) and flag any missing or inconsistent items.
- Create a concise missing-information checklist (e.g., business expense detail, support documentation for dependents, residency dates).
- Identify obvious mismatches between source documents and return entries and record them for prioritized review.
- Note any client assertions that require verification (support provided, living arrangements, filing status choice).

2. Plan & Structure
- Map each discrepancy to the specific tax forms/schedules that could change (income, adjustments, schedule(s), credits, and state carryovers).
- Enumerate the federal and California rule areas to review and cite (income reporting, dependent/credit eligibility, itemizing vs standard deduction, self-employment tax, state conformity deviations).
- Define assumption protocols for missing data (conservative/default options, sensitivity ranges) and document how estimates will be presented.
- Sequence the recalculation workflow: reconcile gross income, determine adjustments, compute AGI, decide itemize vs standard, recalc tax/credits, compute self-employment tax, then re-run state return using corrected federal figures.
- Prepare a list of forms/schedules likely to need revision so drafts can be generated in order.

3. Draft -> Self-Review -> Finalize
- Produce corrected worksheets and draft forms, annotate each change with source and rationale.
- Run arithmetic and cross-form consistency checks; verify federal-to-state data flow.
- Re-evaluate dependent and filing-status determinations using a checklist; document any remaining uncertainties.
- If exact liability cannot be determined, provide an explicit list of missing items, present a documented estimation framework with ranges, and flag items requiring client follow-up.
- Compile a final action list and a client-ready summary of changes needed and next steps.
