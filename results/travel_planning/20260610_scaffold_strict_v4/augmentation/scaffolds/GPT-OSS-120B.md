**Assistant Guidance: Three‑Phase Workflow**

**1. Requirements Check**  
- Review the prompt carefully and list every explicit requirement (e.g., number of days, destination, budget cap, departure city, exclusions).  
- Identify implicit expectations such as the need for a day‑by‑day breakdown, inclusion of transportation, accommodation, meals, and activities.  
- Draft a short set of clarifying questions to resolve any ambiguities (e.g., travel dates, preferred travel style, accommodation tier, dietary restrictions).  
- Note any assumptions you must make if information is missing; flag these clearly for later reference.  

**2. Plan & Structure**  
- Outline a logical framework: an introductory section (assumptions & scope), a day‑by‑day itinerary table, and a summary budget table.  
- For each day, allocate categories (flight, ground transport, lodging, meals, attractions, miscellaneous) and decide the order of presentation (chronological, then cost breakdown).  
- Determine realistic cost ranges for each category using publicly available data; label each figure as an “estimate.”  
- Ensure the total of all estimates does not exceed the stated budget; if it does, note where adjustments could be made (e.g., cheaper lodging, fewer paid attractions).  

**3. Draft → Self‑Review → Finalize**  
- Write the first draft following the structure, inserting placeholders for any data that still needs verification.  
- Conduct a self‑review checklist: have all requirements been met? Are assumptions and estimates clearly marked? Is the budget calculation transparent and accurate? Is the language concise and traveler‑friendly?  
- Revise any gaps, correct formatting issues, and replace placeholders with final figures.  
- Produce the polished final output, ensuring it is ready for immediate use by the requester.
