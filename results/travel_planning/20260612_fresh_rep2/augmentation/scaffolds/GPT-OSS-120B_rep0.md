**Assistant Guidance: Three-Phase Workflow**

**1. Requirements Check**  
- Re‑read the prompt carefully and list every explicit requirement (e.g., number of days, budget cap, departure city, need for assumptions, need for clarifying questions).  
- Identify any implicit constraints (e.g., realistic cost categories, need for labeling estimates).  
- Compile a short “requirements checklist” and verify each item is addressed before moving on.  
- Draft a set of clarifying questions that resolve any ambiguities (travel dates, preferred travel style, accommodation type, etc.).  

**2. Plan & Structure**  
- Outline the overall structure of the response: introductory assumptions, clarifying‑question section, day‑by‑day itinerary, cost breakdown, and concluding notes.  
- Decide on consistent formatting (e.g., bullet points for daily activities, a table for budget items, bold headings for each day).  
- Determine where to place labels such as “estimate” and how to present totals so the budget constraint is transparent.  
- Sketch placeholder headings and sub‑headings, ensuring each requirement from Phase 1 maps to a specific section.  

**3. Draft → Self‑Review → Finalize**  
- Write a first draft following the outline, inserting generic placeholders where specific details will later be filled.  
- Conduct a self‑review using the checklist: confirm all requirements are met, verify that assumptions are clearly stated, ensure cost figures are marked as estimates, and check for logical flow and readability.  
- Revise any gaps, tighten language, and format consistently.  
- Perform a final proofread for grammar, punctuation, and adherence to the word‑count guideline, then deliver the polished itinerary.
