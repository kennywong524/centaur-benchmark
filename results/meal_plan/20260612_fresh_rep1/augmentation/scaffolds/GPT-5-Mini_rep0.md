**Assistant Guidance: Three-Phase Workflow**

1. Requirements Check
- Read the assignment carefully and extract all constraints: dietary restrictions, intolerances, dislikes/likes, budget, simplicity, and format deliverables (7 days, 3 meals + snacks, grocery list, prep notes).
- Create a checklist of "must-not-include" items and a separate list of "must-include" elements (meals per day, snack types, shopping categories, prep notes).
- Clarify assumptions to be validated (portion sizing approach, cost ceiling, available kitchen equipment, time per meal).
- Identify verification steps and resources for allergen safety and ingredient substitutes (avoid relying on a single source).

2. Plan & Structure
- Define a reusable daily template (breakfast, lunch, dinner, snack 1, snack 2) and a consistent format for each day entry.
- Set constraints for variety and balance across the week (ingredient rotation, protein sources, starches, flavors) without specifying actual foods.
- Design the grocery list layout: grouped by store section, quantities by week, pantry vs perishables, optional substitutions.
- Decide how to present brief prep notes: total time, make-ahead tips, batch-cook opportunities, reheating/storage guidance.
- Allocate space for allergy cross-contamination warnings and seasoning adjustment notes (kept generic).

3. Draft -> Self-Review -> Finalize
- Produce a first draft following the template, filling each slot with compliant meal placeholders.
- Run a self-review checklist: adherence to dietary exclusions, no forbidden ingredients mentioned, budget/simplicity met, variety, clear grocery quantities, and prep clarity.
- Validate readability and user actionability; adjust phrasing for conciseness.
- Finalize by formatting consistently, adding the grocery list and concise prep notes, and flagging any assumptions for user confirmation.
