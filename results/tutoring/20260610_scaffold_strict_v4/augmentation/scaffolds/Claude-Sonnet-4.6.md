**Assistant Guidance: Three-Phase Workflow**

---

**Phase 1: Requirements Check**

Before writing anything, identify and list all mandatory components explicitly requested in the task. Confirm the target audience constraints (age, grade level, cognitive stage). Note any format or pedagogical requirements (visual support alongside notation, age-appropriate language). Flag any components that could conflict or require special handling (e.g., balancing mathematical accuracy with simplified language). Clarify the scope: this is a *segment*, not a full lesson plan, so calibrate depth accordingly.

---

**Phase 2: Plan & Structure**

Map out a logical sequence for the segment's components so each builds on the previous one. Consider a natural pedagogical flow: objective → concept introduction → analogy → worked examples → misconception awareness → activity → comprehension check → wrap-up. Decide how visual support will be referenced or described throughout (since abstract notation must be paired with visuals). Anticipate how interactive elements will be framed for a classroom setting. Ensure the misconceptions section connects back to the examples rather than standing alone.

---

**Phase 3: Draft → Self-Review → Finalize**

Write a complete first draft covering every required component in sequence. Then self-review against this checklist:
- Is every component from Phase 1 present?
- Is language consistently age-appropriate throughout?
- Does every instance of notation include a described visual?
- Are transitions between sections smooth and logical?
- Is the wrap-up concise and tied to the learning objective?

Revise any gaps or inconsistencies, then finalize.
