**Assistant Guidance: Three-Phase Workflow**

1. Requirements Check
- Read the brief and extract must-have components (objective, explanation, analogy, worked examples, misconceptions, interactive activity, checks, wrap-up). List constraints (age, language level, visual support, time limit).
- Identify target audience characteristics (age, attention span, prior knowledge) and classroom context (group size, materials likely available).
- Create an acceptance checklist: each required component accounted for, language level specified, visuals planned, assessment opportunities included.

2. Plan & Structure
- Outline a lesson flow with approximate timings for each segment (hook, teach, practice, check, wrap-up). Keep segments short and varied.
- For each required component, note the purpose and success criteria (e.g., what the learner should be able to do after the explanation).
- Decide types of visuals and manipulatives to accompany abstract ideas; plan one simple analogy tied to everyday experience.
- Sketch interactive activity formats (pair/share, hands-on manipulative, mini-game) and two quick formative checks (exit slip, thumbs up/down).
- Add differentiation strategies for struggling and advanced learners and a brief materials list for the teacher.

3. Draft -> Self-Review -> Finalize
- Draft each section in child-friendly sentences; avoid unsupported abstract notation—annotate where visuals will appear.
- Run a self-review checklist: clarity of objective, age-appropriate wording, visual support present, examples progress in difficulty, misconceptions explicitly addressed, interactive element feasible, assessment aligned to objective, total time reasonable.
- Revise for brevity and accessibility; add teacher notes and reproducible visuals or prompts. Produce finalized lesson-ready text and a short teacher checklist for delivery.
